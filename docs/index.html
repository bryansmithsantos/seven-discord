Project History

Seven-Discord was born from a frustration with the bloated state of modern Discord libraries. While libraries like discord.js are powerful, they are heavy and memory-intensive.

Inspired by Aoi.js (a string-based bot package) but wanting the type-safety and speed of Bun, we created Seven.

  • v1.0: A simple proof-of-concept macro interpreter.
  • v2.0: Complete rewrite of the logic engine for nested macros.
  • v2.5 (The Mega Expansion): Added a full Economy system, extensive System macros, and completed the verification of all Logic/Math modules.
