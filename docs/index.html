Project History

Seven-Discord began as an experiment to simplify Discord bot development using Bun's incredible performance.

Unlike discord.js, which is feature-complete but heavy, Seven focuses on raw speed and a unique "Macro" system inspired by Aoi.js but modernized for TypeScript.

v1.0: Initial prototype.
v2.0: Macro Interpreter overhaul.
v2.5: The Mega Expansion (Current), adding 90+ macros, full economy, and bilingual support.
