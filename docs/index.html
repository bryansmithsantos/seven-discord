Seven-Discord

O framework de macros mais rápido e simples para criar bots no Discord. Feito para Bun.

Ultra Rápido

Otimizado para Bun. Sem overhead.

Sistema de Macros

Lógica complexa em uma linha.

Bot Verificado

Slash, Botões, Modais prontos.

Quick Start

# Instalar
bun add seven-discord

# Criar projeto
bunx seven-discord --init --name meu-bot

# Rodar
bun run dev

A Origem

Por que Seven?

Por muito tempo, criar bots era sinônimo de boilerplate infinito. O Seven nasceu da frustração com a complexidade desnecessária. Uma linha de código deveria fazer uma coisa. Simples assim.
