Moderation Macros

Module

Keep your server safe. Punish rule breakers.

Ban & Kick

s.ban[UserID; Reason]

Bans a user from the server.

s.kick[UserID; Reason]

Kicks a user from the server.

Timeouts (Mute)

s.mute[UserID; Duration; Reason]

Times out a user. Duration example: 1h, 30m, 1d.

s.unmute[UserID; Reason]

Removes the timeout.

Warnings System

s.warn[UserID; Reason]

Warns a user. Stores in database.

s.unwarn[UserID; WarnIndex]

Removes a specific warning.

s.warnings[UserID]

Lists all warnings for a user.

Channel Control

s.lock

Prevents @everyone from sending messages.

s.unlock

Allows @everyone to send messages.

s.purge[Amount]

Deletes X messages in current channel.

s.slowmode[Seconds]

Sets channel rate limit (0 to disable).
