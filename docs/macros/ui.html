Module

Build rich interfaces with Embeds, Buttons, and Menus.

s.embed

s.embed[InnerMacros...]

Creates an Embed. Use helpers inside.

s.embed[
    s.title[Hello]
    s.desc[This is an embed]
    s.color[#FF0000]
]

s.onInteraction

s.onInteraction[ID; Code]

Registers code to run when a component with ID is used.

    s.reply[You clicked it!]
