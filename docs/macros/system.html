System Macros

Module

Bot and Server information utilities.

s.ping

Returns "Pong!". Simple latency check.

s.uptime

Returns how long the bot has been online.

s.botInfo

Displays RAM usage, server count, and command count.

s.serverInfo

Displays info about the current guild (Members, ID, Owner).

s.userInfo[UserID]

Displays info about a user (Creation date, Joined date).

s.avatar[UserID]

Returns the avatar URL of a user.

s.eval[Code] DANGER

Executes arbitrary Javascript code. Use with EXTREME caution. Only for owners.
